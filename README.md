# KodPlugins

自制的可道云插件，部分插件暂未支持新版的 `KodBox`，正在迁移中。



## 插件安装方法

下载后，将对应版本的插件文件夹上传至可道云的 `plugins` 文件夹下，在插件中心启用插件并刷新页面即可！
